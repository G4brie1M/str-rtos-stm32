# Drivers

This directory contains the official STM32 software drivers used by the project.

The contents of this directory are primarily provided by STMicroelectronics and include:

- CMSIS core support files
- STM32 HAL (Hardware Abstraction Layer) drivers
- Device-specific peripheral definitions

The original structure should be preserved whenever possible.
