# STM32G4 HAL Driver

This directory contains the STM32 Hardware Abstraction Layer (HAL) implementation.

The HAL provides software interfaces for:

- GPIO
- Timers
- ADC
- UART
- DMA
- Interrupts
- Other STM32 peripherals

These files are generated and maintained by STMicroelectronics.

Project-specific application code should not be added directly to this directory.
